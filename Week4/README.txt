Image_Processing.py :  입력 이미지 파일에서 범위를 지정(차량의 시점으로 하단 부분 시점 설정 --> 이미지 640*480의 경우)후 이미지에서 차선 검출

image.py : 사용법[python image.py ./image/red_line1.jpg true true], 입력 이미지에서 빨간색 라인을 탐지 합니다.
